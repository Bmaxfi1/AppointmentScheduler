This is an application in which users can create, update, and delete appointments and customers.
To run the application, double click the .jar file in out/artifacts/AppointmentScheduler/AppointmentScheduler.jar

The first report is accessed via the reports tab.
The second report is accessed by searching the name or Id of the contact using the search button on the 'all
Appointments' tab.
The third report is a list of all customers in a specific first level division.

MySQL driver version = mysql-connector-java-8.0.22
